# 一键清屏 (One-click Clear Screen)
![language Python](https://img.shields.io/badge/language-Python-blue.svg)
## 简介

这是一个为 Windows 用户设计的工具，旨在实现一键关闭任务栏中所有运行的软件进程。它解决了 Windows 10/11 系统未提供一键关闭所有软件功能的痛点。

## 功能特点

- **强力关闭模式**：采用两阶段关闭逻辑。第一阶段发送关闭信号（WM_CLOSE），第二阶段自动强制终止（Kill）仍未退出的顽固进程。
- **广泛兼容**：放宽了主窗口识别标准，能够有效处理如“Legion Zone”等采用非标准窗口样式的应用。
- **安全保护**：自动排除系统核心组件及内核进程，确保系统稳定。
- **操作简便**：无需安装，直接执行。
- **静默运行**：执行时无多余窗口，完成后自动退出。

## 使用说明

1. 下载并运行 `dist/close_all_apps.exe`。
2. 程序会自动向所有可见的应用程序窗口发送关闭信号。
3. 建议在需要快速清理工作台或准备关机前使用。

## 技术细节

- **核心逻辑**：通过枚举所有顶层窗口，识别具有可见标题且非系统工具窗口的应用，并发送 `WM_CLOSE` 消息。
- **开发语言**：Python
- **依赖库**：`pywin32`, `psutil`
- **编译工具**：PyInstaller
